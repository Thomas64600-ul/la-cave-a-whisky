# Frontend – La Cave à Whisky 

Ce dossier contient la partie **interface utilisateur** du projet.

## Structure

frontend/
├── index.html → Page d'accueil (API Whisky Edition)
├── cave.html → Cave de Bon Papa
├── details.html → Fiche d’un whisky
├── add-whisky.html → Formulaire d’ajout
├── tasting.html → Page de dégustations
├── login.html → Connexion
├── register.html → Inscription
├── components/ → Header, footer, cartes réutilisables
├── assets/ → Images et CSS
├── js/ → Scripts JavaScript
└── README.md


## Technologies
- HTML5
- CSS3 (Grid / Flexbox)
- JavaScript ES Modules
- Mode clair / sombre
- API Whisky Edition + Backend Bon Papa
